Le script ultime.bat permet de jouer un coup complet.

- Le grille de jeu est décrite dans le fichier grille.txt.
- 3 joueurs jouent et ont saisi les mots qu'ils ont respectivement trouvés dans coup-j1.txt, coup-j2.txt et coup-j3.txt.
- le dictionnaire considéré est ods4.txt

Le script affiche le nombre de points remportés par chaque joueur ainsi que le maximum de points pouvant être gagnés :

8 point(s) pour le joueur 1
13 point(s) pour le joueur 2
2 point(s) pour le joueur 3
----------
431 point(s) au maximum possible

Le script fait l'hypothèse que les programmes du projet sont nommé exo1.exe, exo2.exe, ..., exo6.exe et qu'ils sont tous dans un même répertoire.

Pour pouvoir vous en servir, vous devez éditer le fichier ultimate.bat (ouvrez le avec un éditeur de texte) pour indiquer dans quel répertoire se trouvent les exécutables et dans quel répertoire se trouve ods4.txt.

Une fois configuré, vous pouvez vous servir du script en tapant son nom à l'invite de commande.

Le script timecmd.bat vous permet de connaitre le temps d'exécution du script ultime.bat. La commande à saisir est :

timecmd ultime

Sur mon ordinateur (un pc classique) et avec mes programmes, l'exécution de la commande prend moins de 3 secondes (2,3 sec exactement).
